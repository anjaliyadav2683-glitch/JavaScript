What is JavaScript?

JavaScript is a high‑level, dynamic programming language used to make web pages interactive.

Why JavaScript?

  • It powers most modern websites.
  • It is beginner‑friendly yet extremely powerful.
  • It works on both frontend and backend (with Node.js).
  • It has a huge ecosystem of libraries and frameworks.

Where Does JavaScript Run?

  • In the browser (Chrome, Firefox, Edge, etc.)
  • On servers using Node.js

Variable

What is a Variable in JavaScript?

A variable is like a box where you store something (data) so you can use it later.

Just like a box with a label.

Why Do We Need Variables?

Because programs need to store information like:

  • user name
  • age
  • marks
  • price

How to Create a Variable

  • let
  • var
  • const

What is var?

A function-scoped variable declaration (older method).

What is let?

A block-scoped variable that can be reassigned.

What is const?

A block-scoped variable that cannot be reassigned.

(1) let → Modern & Most Used

(2) const → Fixed Value

(3) var → Old Way (avoid mostly)
